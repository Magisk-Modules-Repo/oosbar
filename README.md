# Oxygen OS Gesture Bar for AOSP

This module will change the Stock Android 10 gesture bar (when using the android 10 gestures) to the larger Oxygen OS one
## Requirements
- Android 10 only
- Magisk 19+

## Installation
1. Flash this module.
2. Reboot.
3. Enjoy!
