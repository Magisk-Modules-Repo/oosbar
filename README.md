# Oxygen OS Gesture Bar for AOSP

This module will change the Stock Android 10-11 gesture bar to the larger Oxygen OS one
## Requirements
- Android 10-11 only
- Magisk 20+

![](https://i.imgur.com/3GIKuoE.jpg)

## Installation
1. Flash this module.
2. Reboot.
3. Enjoy!
